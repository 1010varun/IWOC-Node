About IWOC

Innogeeks Winter of Code is an online program focused on bringing new contributors into open source software development.
This event is aimed to develop an open-source culture in KIET Group of Institutions.
It will be a month-long online event, in which the participants can contribute to various projects shortlisted by us, wherein the top performers will be awarded with swags and goodies.

Frequently asked questions

How many team members can I invite?

Students of KIET Group of Institutions (we plan to extend next edition onwards).

Do i need to be good in development for taking part?

No! You can be a beginner and still contribute, we're there to help you out.

How to contribute?

The project links are availabe on the site. One can use their github id for making the contributions

Is the event online/offline?

Online.

Can I participate in IWoc as both a maintainer and a IWoc Contributor?

A maintainer is expected have a understanding of GitHub and Git Version Control System.

Are participants from other colleges allowed to participate in IWoC 2023?

No
